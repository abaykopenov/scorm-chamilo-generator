## Overview

SCORM 1.2 Generator for Chamilo — веб-сервис на Next.js 15 (App Router), который генерирует SCORM 1.2 совместимые обучающие пакеты для LMS Chamilo. Поддерживает AI-генерацию контента через локальные LLM (Ollama, OpenAI-compatible), RAG (Retrieval-Augmented Generation) с использованием векторного поиска (Qdrant / локальные JSON-векторы), и автоматическую публикацию в Chamilo.

## Data Flow

1. **Создание курса:** UI (CourseCreator) → `/api/courses/generate` → `course-generator.js` → [LLM + RAG опционально] → `course-store.js` (сохранение JSON)
2. **RAG pipeline:** Upload файлов → `document-parser.js` → `chunker.js` → `embeddings.js` → `material-indexer.js` → vector store (Qdrant/JSON)
3. **Экспорт SCORM:** `/api/courses/[id]/export-scorm` → `lib/scorm/exporter.js` → manifest + runtime + HTML → ZIP
4. **Публикация:** `/api/courses/[id]/publish-chamilo` → `chamilo-client.js` → авторизация → загрузка ZIP
