## 💭 Идеи на будущее

### Поддержка Moodle
- Интеграция через Moodle Web Services API (REST/JSON)
- Создание Quiz через `core_question_create_category` + `mod_quiz_add_question`
- Загрузка SCORM через `mod_scorm_insert_scorm_data_base`
- **Сложность:** Высокая (Moodle API более структурированный, но сложнее Chamilo)

### Мультиязычная генерация
- Параметр `outputLanguage` в промптах
- Автоопределение языка PDF через первые 500 символов
- Перевод курса на другой язык (промпт: "переведи на английский, сохрани структуру")

### Генерация из нескольких PDF
- Индексация нескольких документов в один RAG-коллекцию
- Генерация курса, комбинирующего факты из всех источников
- Указание приоритетов документов

### Dashboard аналитики
- Страница `/admin/dashboard`
- Статистика: курсов создано, среднее время, средний балл
- Графики: генерация по дням, распределение качества
- Лог ошибок LLM

### SCORM 2004 поддержка
- Поддержка sequencing & navigation
- Более продвинутая аналитика прохождения
- **Сложность:** Очень высокая (SCORM 2004 = сложный стандарт)

### Генерация изображений для слайдов
- Интеграция с Stable Diffusion / DALL-E
- Автоматическая генерация иллюстраций по тексту слайда
- Сохранение в SCORM-пакет

### Экспорт в другие форматы
- PDF (конспект курса)
- PowerPoint (PPTX)
- Markdown (для GitHub)
- xAPI (Experience API) для продвинутой аналитики

### Автоматическое тестирование качества
- E2E тесты: PDF → генерация → проверка структуры
- Smoke тесты: загрузка в Chamilo → прохождение теста → проверка балла
- CI/CD pipeline для автотестов
