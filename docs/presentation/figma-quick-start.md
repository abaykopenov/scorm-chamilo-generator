# Figma Quick Start

1. Создайте Figma frame `1920x1080`.
2. Подключите шрифты: `Sora`, `Manrope`, `IBM Plex Mono`.
3. Импортируйте токены из `figma-tokens.json` вручную или через плагин Tokens.
4. Примените grid: 12 колонок, margin 120, gutter 24.
5. Используйте структуру из `figma-design-spec.md`.
6. Вставьте текст слайдов из `figma-slide-content-ru.md`.
7. На архитектурные слайды добавьте изображения:
   - `docs/architecture/architecture-large.png`
   - `docs/architecture/architecture-1-client-api.png`
   - `docs/architecture/architecture-2-rag-pipeline.png`
   - `docs/architecture/architecture-3-data-delivery.png`
8. Экспортируйте в PDF/PPT по необходимости.
